## 2. 创建 prompt_refine.py Gradio 页面

位置：`dataflow-agent/gradio_app/pages/prompt_refine.py`

功能：
- **左侧**：测试数据编辑（JSONL 格式），可加载/保存文件
- **右侧**：多轮对话调整 prompt 模板
- **下侧**：运行测试按钮，遍历 JSONL 数据调用 LLM

### 关键函数
- `modify_prompt_with_llm()`: 调用 LLM 根据用户反馈修改 prompt
- `strip_think_tags()`: 解析 LLM 输出，分离 <think> 标签内容和实际回复
  - 返回 `(think_content, final_content)` 元组
- 多轮对话时会从 `prompt_template` 输入框获取当前值，调用 LLM 修改
