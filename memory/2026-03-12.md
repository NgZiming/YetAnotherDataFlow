### 2. 创建 FlowUsPromptGenerator Operator

**文件：** `dataflow-extensions/dataflow_extensions/operators/infographic/flowus_prompt_generator.py`

**功能：** 从 FlowUs JSON 模板目录生成 prompt

**核心特性：**
- 从目录随机选择 JSON 模板
- 从 dataframe 行的 `input_topic_key` 字段获取主题
- 批量处理模式（一次性发送所有请求）
- 模板和宽高比配置在初始化时加载，避免重复 IO
- 支持中英文生成

**参数设计：**
- `__init__` 参数：`llm_serving`, `flowus_json_dir`, `aspect_ratios_file`, `seed`
- `run` 参数：`input_topic_key`, `output_key`, `language`, `count`

**迭代过程：**
1. 初始版本：`topics_file` 参数读取 `topics_cc.json`
2. 改为从行的 `input_topic_key` 获取主题
3. `seed` 移到 `__init__` 避免重复随机
4. 改为批量处理模式提高效率
5. 模板在初始化时加载
6. `aspect_ratios_file` 也在初始化时加载
7. 移除默认宽高比，不存在时直接报错
8. 添加 LLM 角色定义系统提示

**Commits:**
