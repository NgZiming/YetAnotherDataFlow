### OpenClaw WebSocket 客户端开发

**任务**: 学习 OpenClaw webchatui 的 WebSocket 通信机制，并用 Python 实现一个可复用的客户端组件。

**主要成果**:

1. **分析 webchatui 源码**
   - 定位到 `control-ui/assets/index-sc9GrLxu.js` 中的 WebSocket 客户端实现
   - 核心类 `Jt` 管理连接状态和消息收发
   - 消息格式:
     - 请求:`{"type": "req", "id": "uuid", "method": "...", "params": {...}}`
     - 响应:`{"type": "res", "id": "uuid", "ok": true/false, "payload": {...}}`
     - 事件:`{"type": "event", "event": "...", "seq": N, "payload": {...}}`
   - 认证流程: `connect.challenge` → 发送带签名的 `connect` 请求 → `connect.hello`

2. **SSL/TLS 安全机制**
   - `isSecureWebSocketUrl()` 函数验证 WebSocket URL
   - 只允许 `wss://` 或本地 `ws://` 连接
   - 支持设备身份认证 (Ed25519 签名)

3. **Python 客户端实现** (`openclaw_webchat_simulator.py`)
   - 类名:`OpenClawGatewayClient`
   - 支持 WSS/HTTPS 连接
   - `verify_ssl=False` 可连接自签名证书
   - 支持自定义 CA 证书
   - 事件回调机制:`on_message`, `on_timeline`, `on_connected`, `on_disconnected`, `on_error`
   - 自动重连功能 (可选)
   - 数据类:`ChatMessage`, `TimelineMessage`, `GatewayConfig`

**使用方法**:
```python
from openclaw_webchat_simulator import OpenClawGatewayClient

client = OpenClawGatewayClient(
    url="wss://gateway.example.com",
    token="xxx",
    verify_ssl=False  # 自签名证书
)

client.on_message = lambda msg: print(f"{msg.sender}: {msg.text}")
await client.connect()
await client.send_chat_message("你好")
await client.disconnect()
```

**命令行工具**:
```bash
python openclaw_webchat_simulator.py --url ws://localhost:8080 --token xxx --insecure --interactive
```

## 技术要点记录

1. **WebSocket 消息协议**
   - 使用 UUID 追踪请求/响应配对
   - 序列号 `seq` 用于检测消息丢失
   - Promise/Future 模式处理异步响应

2. **SSL 上下文配置**
   - 不验证:`ctx.check_hostname = False; ctx.verify_mode = ssl.CERT_NONE`
   - 自定义 CA:`ssl.create_default_context(cafile="ca.crt")`

3. **设计模式**
   - 观察者模式：事件回调
   - 工厂模式:SSL 上下文创建
   - 数据类：消息封装
